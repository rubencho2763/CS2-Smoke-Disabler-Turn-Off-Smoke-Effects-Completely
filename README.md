## 🔍 How it works

The software scans your Counter-Strike 2 folder to locate the particle effect files. These files contain instructions for the game engine on how to draw smoke. The program replaces the texture references with empty paths. When the game tries to load a smoke grenade, it finds an empty file instead of the heavy cloud texture. This removes the visual obstruction from your screen. The changes only affect the visual appearance and do not impact other game mechanics or hitboxes.
